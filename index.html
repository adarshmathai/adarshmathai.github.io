M.Tech Project

Under the Guidance of Prof. Vivek Sangwan | Lab: INDUS Lab, IIT Bombay

Development and Hardware Implementation of Sensor Fusion Algorithm to Improve the Estimation of Aerial Robot Position and Orientation

  • Currently working on the implementation of a computationally efficient and accurate sensor fusion algorithm that obeys the manifold constraints and outputs the optimally accurate data to calculate the position and attitude of aerial robots.
  • Implemented Rotation matrix integration method that stays within the manifold constraints, thus avoiding any singularities that may occur due to parametrization of attitude.
  • Analyzed the noise parameters of IMUs for easier integration of the measured data in the future.
  • Implemented a nonlinear complementary filter (Mahony filter), in Simulink and a microcontroller (teensy 4.0) and analyzed the error and convergence data obtained.
  • Future works include classification and analysis of IMUs, developing a custom filter that operates on rotation matrix (as opposed to other algorithms that operate on its parameters), optimizing the algorithm for the given set of IMUs, and implementing the algorithm in a multirotor aerial robot.
